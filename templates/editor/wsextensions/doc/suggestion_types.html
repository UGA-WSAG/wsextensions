The Service Suggestion Engine (SSE) provides three types of service suggestion.

In order to use the SSE, at least one tool needs to be on the current Workflow Canvas. Tools in the workflow are added to various lists by the user depending on the type of suggestion chosen and the roles these tools should take. The sections below provide information about the three different types of suggestion and their corresponding lists.

Forward Suggestion

When a "Forward Suggestion" is chosen, the SSE will suggest a list of available tools, ranked according to how well they match some desired functionality and their compatibility with the outputs of tools chosen from the Workflow Canvas. The user chooses these Tools to Feed From by selecting them from the corresponding drop-down menu that appears in the "Setup & Run Query" panel to the right of the Workflow Canvas. In the image to the right, the empty box represents the Tools to feed From and the question mark (?) box represents tools that the user would like suggested.

Backward Suggestion

When a "Backward Suggestion" is chosen, the SSE will suggest a list of available tools, ranked according to how well they match some desired functionality and their compatibility with the inputs of tools chosen from the Workflow Canvas. The user chooses these Tools to Feed Into by selecting them from the corresponding drop-down menu that appears in the "Setup & Run Query" panel to the right of the Workflow Canvas. In the image to the right, the empty box represents the Tools to feed Into and the question mark (?) box represents tools that the user would like suggested.

Bidirectional Suggestion

When a "Bidirectional Suggestion" is chosen, the SSE will suggest a list of available tools, ranked according to how well they match some desired functionality and their compatibility with the inputs and outputs of certain tools chosen from the Workflow Canvas. The user chooses Tools to Feed From and Tools to Feed Into by selecting them from the corresponding drop-down menus that appears in the "Setup & Run Query" panel to the right of the Workflow Canvas. In the image to the right, the left box represents the Tools to feed From, the right box represents the Tools to feed Into, and the question mark (?) box represents tools that the user would like suggested.
